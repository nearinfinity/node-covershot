What is Covershot?

Covershot is a multi-format, testing framework agnostic that generates a coverage report for the set of tests for your code.

Download .zip Download .tar.gz View on GitHub

Require

Use covershot's require in your unit tests to display the different parts you want in the report.

var csrequire = require('covershot').require.bind(null, require);
          var myLibrary = csrequire('../lib/myLibrary');

JSCoverage

Instrument your code using JSCoverage to measure the coverage.

jscoverage --no-highlight lib lib-cov

Run Tests

Run each unit test to be reported on.

npm test

Generate Code Metrics

Create metrics for the measurement of each property.

./node_modules/node-jsmeter/bin/jsmeter.js -o ./covershot/jsmeter/ ./lib/

Coverage Report

Generates a coverage report for the code tha you are currently testing to show the how throughly it is being tested.

Installation

$ npm install covershot
